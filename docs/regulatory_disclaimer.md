## Project Name

Glucose Trend Alert System

## Disclaimer

The Glucose Trend Alert System is an educational and portfolio software project.

This project uses synthetic data only.

This project is not a medical device.

This project is not intended for clinical use.

This project should not be used to diagnose, treat, prevent, or manage diabetes or any medical condition.

This project should not be used for medical decision-making.

This project does not connect to any real continuous glucose monitor.

The intended purpose of this project is to demonstrate:

- Python software engineering
- Synthetic healthcare data simulation
- Time-series analysis
- Glucose trend detection
- Simulated alert generation
- Dashboard development
- Data pipeline design
- Testing and validation documentation
- Medical-device software awareness
